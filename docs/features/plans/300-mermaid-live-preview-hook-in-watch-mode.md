# Implementation Plan: Mermaid Live-Preview Hook in Watch Mode

## Overview
Extend `watch` and `sync` commands to export Mermaid C4 diagrams automatically to `architecture.md` on every model change. This enables instant architecture preview in GitHub/GitLab browsers without draw.io.

## Phase 3: Watch Mode Integration â€” Post-Sync Hook

### 3.1 Extend `internal/watcher/watcher.go`
- Add parameters to `Run()`:
  - `enableMermaid bool`
  - `mermaidOutput string`
- After each successful `sync.Run()` in the watch loop:
  - If `enableMermaid` is true AND sync succeeded:
    - Export all views to Mermaid diagrams
    - Wrap in Markdown via `markdown.WrapDiagramsInMarkdown()`
    - Write to `mermaidOutput` file
    - Log success: "âś… Exporting Mermaid... architecture.md (3 views)"
  - If sync failed, skip Mermaid export (don't update file)
