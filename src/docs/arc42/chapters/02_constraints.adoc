= Constraints

== Technical

_(Q-3.2.1)_

[cols="1,3", options="header"]
|===
| Constraint | Source

| *Go 1.25+* — the binary is implemented in Go and built with the
toolchain version pinned in `go.mod`.
| `go.mod:3` (`go 1.25.8`)

| *CGO disabled for releases* — produces a single statically linked
binary; required for clean cross-compilation.
| `Makefile:23-46`, `.goreleaser.yml:18-19`

| *Cross-compilation matrix*: linux, darwin, windows × amd64, arm64.
| `Makefile:21-46`, `.goreleaser.yml:20-25`,
`.github/workflows/go.yml:13-25`

| *Four direct dependencies only*: `beevik/etree` (XML),
`fsnotify/fsnotify` (watch), `spf13/cobra` (CLI),
`pgregory.net/rapid` (property-based tests).
| `go.mod:5-9`

| *No runtime dependencies* — no draw.io required for `sync`; only
`export` shells out to the draw.io CLI.
| `internal/export/export.go:32-45` is the only external invocation.

| *No network access at runtime* — explicitly stated and
verified by the C4-PlantUML stdlib include strategy.
| `README.adoc:194-196`, `internal/diagram/diagram.go:19-21`,
OQ-7 team answer.
|===

== Conventions

* `*_test.go` colocated with code _(`CONTRIBUTING.md`)_.
* Property-based tests with `pgregory.net/rapid` where useful
  _(`internal/drawio/label_property_test.go`)_.
* Six-step checklist for new model fields (struct, JSON Schema,
  patchops, reverse-sync, comment-preservation test, roundtrip test)
  _(`CONTRIBUTING.md:9-22`)_.
