== Level 1 responsibilities

[cols="1,3", options="header"]
|===
| Container | Responsibility / key entry points

| `cmd/bausteinsicht`
| Cobra root + one file per command. Wires `--format`, `--model`,
`--template`, `--verbose`. Validates path containment (SEC-001/016).
Entry: `NewRootCmd` _(`cmd/bausteinsicht/root.go:13-65`)_.

| `internal/model`
| `Load`/`Save`/`PatchSave`/`PatchInsert` on JSONC,
`ValidateWithWarnings`, `Resolve`, `FlattenElements`,
`MatchPattern`, `ResolveView`. JSONC stripping handles BOM,
`//`/`/* */`, trailing commas
_(`internal/model/loader.go:155-227`)_.

| `internal/drawio`
| `LoadDocument`/`SaveDocument`/`NewDocument`, `Page` API,
`TemplateSet` parsing, element/connector/label CRUD, `ParseLabel`
and `GenerateLabel` (round-trip property-tested).

| `internal/sync`
| `Run` (`engine.go`), `DetectChanges` (`diff.go`),
`ApplyForward`/`ApplyReverse`, `ConflictResolver` /
`ModelWinsResolver` (`conflict.go`), layout engine (`layout.go`),
metadata/legend boxes (`metadata.go`), `SyncState` persistence
(`state.go`), patch helpers (`patchops.go`).

| `internal/watcher`
| `Watcher` with 300 ms debounce, mutex-serialised `OnChange`.

| `internal/export`
| `DetectDrawioBinary`, `BuildExportArgs`, `ExportPage`,
`SafeViewKey`, `OutputFileName`. `platform_*.go` for OS-specific
draw.io install paths.

| `internal/diagram`
| `FormatView` for PlantUML and Mermaid C4 output, with C4 level
auto-detection and view-aware relationship filtering.

| `internal/table`
| Render view elements as AsciiDoc or Markdown tables.

| `templates`
| `go:embed` of `default.drawio` and `sample-model.jsonc` for `init`.
|===

== Dependency direction

* `cmd/` depends on every `internal/*` package and `templates`.
* `sync → {model, drawio}`.
* `diagram, table → model`.
* `drawio → etree` only.
* No internal-package import cycles.
