== Sync cycle (UC-2)

[plantuml,seq-sync,svg]
....
@startuml
actor User
participant CLI as "cmd/bausteinsicht"
participant Model as "internal/model"
participant Drawio as "internal/drawio"
participant State as "internal/sync.state"
participant Engine as "internal/sync.Run"

User -> CLI: bausteinsicht sync
CLI -> Model: AutoDetect / Load
CLI -> Model: ValidateWithWarnings
alt validation errors
    CLI --> User: exit code 2
end
CLI -> Drawio: LoadDocument
CLI -> State: LoadState
CLI -> Engine: Run(model, doc, state, templates, newPageIDs)
Engine -> Engine: DetectChanges (3-way diff)
Engine -> Engine: ModelWinsResolver.Resolve(conflicts)
Engine -> Engine: ApplyForward (per view)
Engine -> Engine: ApplyReverse
Engine -> Engine: invisible-element warnings
Engine --> CLI: SyncResult (counts, warnings)
CLI -> Drawio: SaveDocument (atomic)
CLI -> State: SaveState (atomic, sha256)
CLI -> Model: Save (only if reverse changed it)
CLI --> User: report (text or JSON)
@enduml
....

== Watch loop (UC-5)

[plantuml,seq-watch,svg]
....
@startuml
actor User
participant CLI as "watch.go"
participant Watcher as "internal/watcher"
participant FS as "fsnotify"
participant Engine as "internal/sync.Run"

User -> CLI: bausteinsicht watch
CLI -> Watcher: New([model, drawio], 300ms)
loop until SIGINT/SIGTERM
    FS -> Watcher: write event
    Watcher -> Watcher: debounce (300 ms)
    Watcher -> CLI: onChange(file)
    CLI -> Engine: Run(...)
    CLI -> CLI: print result
end
@enduml
....

* Sync execution is serialised by `syncMu`
  _(`internal/watcher/watcher.go:25-26`)_.

== Forward application — per view

[plantuml,seq-forward,svg]
....
@startuml
participant Forward as "ApplyForward"
participant View as "View page"
participant Layout
participant Meta as "metadata.go"

Forward -> Forward: build drillDownLinks
loop per view
    Forward -> View: GetPage("view-<id>")
    alt --relayout
        Forward -> View: clearPageElements
    end
    Forward -> Forward: createScopeBoundary(if scope)
    Forward -> Layout: computeLayout(layered/grid/none)
    Forward -> View: populateNewPage
    Forward -> View: applyChangesToPage
    Forward -> View: populateConnectors
    Forward -> View: reconcileViewPage
    Forward -> View: applyDrillDownLinks
    alt scoped view
        Forward -> View: createBackNavButton
    end
    alt config.metadata
        Forward -> Meta: createMetadata
    end
    alt config.legend
        Forward -> Meta: createLegend
    end
end
@enduml

== Init flow (UC-1)

. Verify no default file exists _(`init.go:38-46`)_.
. Write `architecture.jsonc` and `template.drawio` from embedded
  templates _(`init.go:48-54`)_.
. Load model, create a fresh draw.io document, run a single sync to
  populate it, save with sha256 state.
. Report.

== Failure modes

[cols="1,3", options="header"]
|===
| Trigger | Behaviour

| Invalid model on `sync`
| Validation aborts before any write _(`sync.go:50-62`, #176).

| Corrupt state file
| `LoadState` rejects with checksum mismatch and asks the user to
delete `.bausteinsicht-sync` _(`state.go:71-84`).

| Corrupt draw.io file
| `LoadDocument` rejects before any modification
_(`document.go:30-49`).

| Conflict in a field
| Model wins; warning emitted; cycle continues; exit 0
_(`conflict.go:27-55`).

| Element absent from any view
| Warning, but cycle continues _(`engine.go:66-83`, #183).
|===
