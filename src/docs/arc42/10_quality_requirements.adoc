=== Quality scenarios

. *Atomic concurrent edit.* User edits `architecture.drawio` in
  draw.io while `bausteinsicht watch` is running. The watcher absorbs
  the editor's atomic-rename save, polls the path back into the
  watch set, and triggers a sync. Result: no missed event, no
  duplicate sync.
. *Corrupted state file.* `.bausteinsicht-sync` is truncated by an
  external process. Next sync detects checksum mismatch and refuses,
  pointing the user at the remediation. Alternatively, a zero-byte
  state is treated as missing and triggers a clean re-sync.
. *Orphaned managed shape.* A view is removed from the model.
  `RemoveOrphanedViewPages` deletes the corresponding draw.io page;
  `reconcileOrphanedElements` removes managed shapes whose
  `bausteinsicht_id` is no longer in the model.
. *Headless export at 1×.* `bausteinsicht export --image-format png`
  in a container without GPU works because `BuildExportArgs` does
  not pass `--scale` for the default value of 1.0. At `--scale 2`
  the user must have hardware GPU acceleration.
