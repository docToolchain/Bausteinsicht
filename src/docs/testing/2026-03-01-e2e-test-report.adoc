== Recommendations

=== Priority 1 (must fix before further features)

. *Fix #83* — Scoped cell ID corruption. This is the most impactful bug: it silently corrupts user data on the first sync. Root cause: `extractDrawioRelationships()` in `diff.go` needs to map scoped cell IDs back to element IDs.

. *Fix #85* — Element deletion. Forward sync must handle Deleted changes when views are active (currently only works in legacy no-view mode).

=== Priority 2 (should fix)

. *Fix #86* — JSONC comment preservation. Critical for user experience since comments document architectural decisions.

. *Fix #84* — Scope boundary updates. Scope boundaries must be updated when their underlying element properties change.

=== Priority 3 (nice to have)

. *Fix #87-#91* — CLI polish issues. These affect LLM-friendliness and edge cases but don't block core workflows.
