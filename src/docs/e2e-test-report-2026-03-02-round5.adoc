== Root Cause Patterns

=== 1. Missing Drawio Structure Recovery (#175)

The `LoadDocument` validation (added to fix #140) correctly rejects drawio files without `<diagram>` elements. However, the forward sync can create such files when all views are removed. The fix should either preserve at least one empty diagram or allow `sync` to recreate the drawio structure.

=== 2. Validation-Sync Inconsistency (#176)

The `validate` and `sync` commands apply different levels of validation to view patterns. `validate` checks all include/exclude entries against the element tree, but `sync` resolves patterns silently (non-matching patterns produce empty sets). This gap means `sync` can silently change behavior based on typos in patterns.
